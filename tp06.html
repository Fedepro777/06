1.1- ¿Qué significa CSS?

CSS significa cascading style sheets o Las hojas de estilo en cascada. Nos da la liberdat de crear páginas web más bonitas y mejor decoradas.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

Se utiliza para darle estilo a lo creado en html, el programa utilizando para interpretar el CSS es el browser

1.3- ¿A qué se llama regla de CSS?

El CSS consiste en declarar reglas: Bajo varias condiciones, queremos que ocurran ciertas cosas.

1.4- ¿Cuál es la anatomía de la regla de CSS?

la anatomía se define con un selector, o sea una etiqueta donde dentro vamos a definir unos atributos asi darle más detalles y mejor visual.

1.5- ¿Qué es un selector en CSS?

un selector es una etiqueta. dentro de ella vamos a darle propiedades para conseguir que tengan máa personalización

2.0- Utilizaremos las siguientes propiedades de CSS. Completar la columna de valores posibles para cada propiedad.

propiedad Descripciónth Valores posibles
color Define el color del texto. Los colores en ingles, hexadecimal.
background-color Define el color de fondo de un elemento Los colores en ingles, hexadecimal.
font-size Define el tamaño de la fuente del texto. Se puede utiliza en milimetros(mm) o pixeles (px)
font-style Define el estilo de la fuente del texto (por ejemplo, si es cursiva o normal). italica, normal, oblicua, etc
font-family Define el tipo de letra a utilizar para el texto. cursiva, fantasy, serif, etc
text-align Define la alineación horizontal del texto (izquierda, centro o derecha) centred, left, right
text-decoration Define la decoración del texto (por ejemplo, si está subrayado o tachado).
text-transform Define la transformación del texto (por ejemplo, en mayúsculas o minúsculas).

2.3- Utiliza selectores por etiquetas para aplicar estilos a las etiquetas HTML, como el tamaño de fuente, el color de fondo y el color del texto: Utiliza el selector body para aplicar estilos al cuerpo de la página, como el color de fondo y el tamaño de fuente. body { background-color: #f2f2f2; font-size: 16px; } Utiliza el selector h1 para aplicar estilos al encabezado de la página, como el tamaño de fuente y el color del texto. h1 { font-size: 36px; color: #333; } Utiliza el selector p para aplicar estilos al contenido de la página, como el tamaño de fuente y el color del texto a elección.
